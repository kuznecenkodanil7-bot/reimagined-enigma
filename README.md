# Creepypasta Nightmare — Fabric 1.21.4

Жёсткий хоррор-мод для Minecraft Fabric 1.21.4.

## Что есть

- Частые скримеры: полноэкранный overlay + резкий ванильный звук.
- Сбои в чате: глючные системные сообщения.
- Молнии рядом с игроком.
- Удаление и «коррупция» блоков вокруг игрока.
- Ломающееся окружение без намеренной порчи файлов сохранения.
- Новая сущность `creepypasta:stalker`: человекоподобная фигура без ника, преследует игрока.
- При касании сущность переносит игрока в измерение `creepypasta:nightmare`.
- В измерении нужно добраться до портала, который охраняет сущность.
- GitHub Actions автоматически собирает `.jar`.

## Важно

Мод реально меняет блоки в мире. Тестируй на копии мира.

## Как собрать через GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы проекта в репозиторий.
3. Открой вкладку **Actions**.
4. Запусти workflow **Build Fabric mod**.
5. Готовый `.jar` будет в артефактах workflow.

## Локальная сборка

Нужны Java 21 и Gradle 8.11.1+.

```bash
gradle build --no-daemon
```

Файл появится здесь:

```text
build/libs/creepypasta-nightmare-1.0.0.jar
```

## Версии

- Minecraft: `1.21.4`
- Yarn: `1.21.4+build.8`
- Fabric Loader: `0.16.10`
- Fabric API: `0.119.4+1.21.4`
- Java: `21`

## Настройка частоты событий

Смотри `NightmareManager.java` и `CreepypastaClient.java`:

- `tickOverworldHorror` — шанс молний, удаления блоков, спавна сущности.
- `ClientTickEvents.END_CLIENT_TICK` — частота скримеров.
